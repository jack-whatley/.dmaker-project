HTML Editing

To start with we need to laying out the HTML code, to do this we only need to add a button and a output.

The button element just requires you to type:

<button id="button">Submit</button>

The output element just requires you to type:

<p id="output"></p>

Once you are ready to type these out, press the next button.

You can now enter the code into this box below and press submit to check it, if its correct you will be able to proceed to the next section.

If you need help the code is here:
